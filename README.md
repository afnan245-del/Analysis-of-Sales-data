# Sales and Product Analysis

> This project will analyze the relationship of number of sales with different variables to answer Business questions

### Tool used:
- Python
- Pandas
- Seaborn
- Matplotlib

### Dataset:
- Includes 2824 rows and 25 columns
- Main columns are Sales and Product

### Business Questions
1. Which product category generates the highest revenue?
2. Which country contributes most to total sales?
3. Is there a seasonal trend in sales?
